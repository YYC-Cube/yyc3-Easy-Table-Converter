## 1. 项目介绍

YYC³ Easy Table Converter 是一个功能强大、用户友好的表格数据转换工具，旨在为用户提供多格式表格数据的转换、处理和分析功能。本工具采用现代化的Web技术栈构建，支持多种数据格式之间的无缝转换，并提供高级数据处理、分析和性能优化功能，适用于数据分析师、开发人员和需要处理表格数据的各类用户。

## 6. 安装与运行

### 6.1 前置要求

- Node.js 16.x 或更高版本
- npm 或 yarn 包管理器

### 6.2 安装步骤

1. 克隆项目仓库

```bash
git clone [仓库地址]
cd YYC³-Easy-Table-Converter
```

2. 安装依赖

```bash
npm install
# 或
yarn install
```

3. 启动开发服务器

```bash
npm run dev
# 或
yarn dev
```

4. 访问应用

打开浏览器，访问 `http://localhost:3100`

### 6.3 构建生产版本

```bash
npm run build
# 或
yarn build
```

## 7. 使用指南

### 7.1 数据转换功能

1. **上传文件**：点击上传按钮或拖放文件到指定区域
2. **选择格式**：从下拉菜单中选择目标转换格式
3. **调整设置**：根据需要调整转换选项
4. **执行转换**：点击转换按钮生成结果
5. **下载结果**：将转换后的文件下载到本地

### 7.2 主题设置

使用页面右上角的主题切换按钮可以在以下主题间切换：
- 浅色主题
- 深色主题
- 跟随系统主题

### 7.3 用户设置

点击设置按钮进入设置页面，可以配置：
- **外观设置**：主题模式、字体大小等
- **行为设置**：自动保存、通知提醒等
- **语言设置**：选择界面显示语言

│       │   │   ├── loggerMiddleware.ts # 日志中间件
