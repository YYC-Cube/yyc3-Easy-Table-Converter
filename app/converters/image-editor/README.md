# 图片编辑器功能使用指南

## 功能概述

本图片编辑器提供了多种图片处理功能，包括图片裁剪、添加文字和表情、去水印、擦除不需要的部分等操作，让用户能够方便地编辑和美化图片。

## 主要功能

### 1. 图片上传

- 点击上传区域或拖放图片文件到上传区域
- 支持常见图片格式：JPG、PNG、GIF等
- 图片上传后会自动调整大小以适应编辑界面

### 2. 橡皮擦工具

- 点击工具栏中的「橡皮擦」按钮启用橡皮擦功能
- 在画布上拖动鼠标进行擦除操作
- 擦除效果为透明背景
- 再次点击「橡皮擦」按钮关闭该功能

### 3. 裁剪工具

- 点击工具栏中的「裁剪」按钮启用裁剪功能
- 在画布上绘制裁剪区域
- 确认裁剪后，图片将按照指定区域进行裁剪
- 点击「完成」按钮应用裁剪，或点击「取消」放弃裁剪

### 4. 添加文字

- 点击工具栏中的「添加文字」按钮启用文字添加功能
- 在画布上点击需要添加文字的位置
- 输入要添加的文字内容
- 拖动文字可调整位置
- 再次点击「添加文字」按钮关闭该功能

### 5. 添加表情

- 点击工具栏中的「添加表情」按钮启用表情添加功能
- 在画布上点击需要添加表情的位置
- 选择要添加的表情
- 拖动表情可调整位置
- 再次点击「添加表情」按钮关闭该功能

### 6. 去水印功能

- 点击工具栏中的「去水印」按钮
- 系统会自动检测并移除图片中的水印
- 处理完成后会显示操作结果提示

### 7. 去文字功能

- 点击工具栏中的「去文字」按钮
- 系统会自动检测并移除图片中的文字内容
- 处理完成后会显示操作结果提示

### 8. 撤销/重做功能

- 点击顶部工具栏的「撤销」按钮可撤销上一步操作
- 点击顶部工具栏的「重做」按钮可重做已撤销的操作
- 历史记录最多保存20步操作

### 9. 保存图像

- 点击顶部工具栏的「保存图像」按钮
- 编辑后的图片将以PNG格式下载到本地

## 使用技巧

1. **元素拖拽**：添加文字或表情后，可以通过鼠标拖拽来调整它们的位置
2. **组合操作**：可以结合多种工具使用，如先裁剪图片，再添加文字和表情
3. **撤销操作**：操作失误时，可以使用撤销功能恢复到之前的状态
4. **批量处理**：对于多张图片需要相同处理时，可以重复使用相同的编辑步骤

## 注意事项

1. 为保证编辑性能，上传的图片会被限制在最大1200×1200像素的范围内
2. 去水印和去文字功能基于AI算法，对于复杂背景的水印或文字可能效果有限
3. 请确保您拥有编辑和处理图片的权限，遵守相关法律法规
4. 编辑过程中请勿刷新页面，否则会丢失未保存的编辑内容
5. 为获得最佳编辑体验，建议使用最新版本的Chrome、Firefox或Safari浏览器

## 常见问题

**Q: 上传图片后没有显示预览怎么办？**
A: 请检查图片格式是否支持，尝试刷新页面后重新上传。

**Q: 去水印功能没有完全移除水印怎么办？**
A: 复杂背景下的水印可能需要多次尝试，或结合橡皮擦工具进行手动处理。

**Q: 编辑后的图片下载失败怎么办？**
A: 请检查浏览器权限设置，确保允许文件下载，并尝试使用不同的浏览器。

**Q: 历史记录不够用怎么办？**
A: 系统最多保存20步操作，建议在关键编辑节点后保存图片作为备份。
