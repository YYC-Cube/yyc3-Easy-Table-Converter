## 1. 系统简介

### 1.1 系统概述
YYC³ Easy Table Converter 是一款功能强大、用户友好的数据转换和处理工具，专为解决企业级数据格式转换、跨系统数据迁移和行业特定数据处理需求而设计。系统支持多种数据格式的导入导出，提供灵活的数据映射配置，并具备丰富的行业特定功能。

### 1.2 主要特性
- **多格式支持**：支持 Excel、CSV、JSON、XML、PDF 等多种数据格式
- **跨行业转换**：内置金融、制造、零售等行业的数据转换能力
- **智能映射**：提供可视化的映射规则配置和智能推荐
- **批处理能力**：支持大批量数据并行处理
- **实时监控**：提供转换任务的实时状态监控和进度展示
- **报告生成**：自动生成数据转换报告和分析统计
- **API 集成**：提供全面的 API 接口，方便系统集成
- **安全性保障**：支持数据加密、访问控制和审计日志

### 1.3 应用场景
- 企业间数据交换和系统集成
- 遗留系统数据迁移
- 跨部门报表转换
- 行业标准数据格式转换
- 数据清洗和规范化处理
- 大批量数据批处理和自动化

#### 4.1.2 批量文件上传
1. 点击「文件管理」->「批量上传」
2. 选择「批量文件上传」选项

### 5.2 自定义脚本

#### 5.2.1 JavaScript 脚本
系统支持通过 JavaScript 脚本实现复杂的数据处理逻辑：

1. 在转换设置中点击「添加自定义脚本」
2. 在脚本编辑器中编写处理逻辑
3. 测试脚本功能
4. 保存并应用到转换任务

**示例脚本：数据清洗**
```javascript
function processRecord(record) {
  // 去除字符串字段两端空格
  for (const key in record) {
    if (typeof record[key] === 'string') {
      record[key] = record[key].trim();
    }
  }
  
  // 标准化日期格式
  if (record.date) {
    const date = new Date(record.date);
    record.date = date.toISOString().split('T')[0]; // YYYY-MM-DD 格式
  }
  
  // 计算总价
  if (record.price && record.quantity) {
    record.totalAmount = parseFloat(record.price) * parseInt(record.quantity);
  }
  
  return record;
}
```

#### 5.2.2 脚本库和模板
系统提供常用的脚本模板库，包括：
- 数据清洗脚本
- 数据验证脚本
- 格式转换脚本

## 8. 映射规则管理

### 8.1 创建映射规则

1. 点击左侧菜单的「映射规则」
2. 点击「创建新规则」按钮
3. 填写规则基本信息：
   - 规则名称：易于识别的规则名称
   - 描述：规则的详细描述
   - 源行业和目标行业：选择相关行业
   - 是否设为默认：是否将此规则设为特定转换路径的默认规则

4. 配置映射关系：
   - 手动添加映射字段
   - 使用自动映射功能推荐映射关系
   - 配置字段转换规则和验证条件

5. 保存规则并应用

### 8.2 映射规则类型

系统支持以下类型的映射规则：

#### 8.2.1 字段映射规则
- **一对一映射**：一个源字段映射到一个目标字段
- **多对一映射**：多个源字段映射到一个目标字段
- **一到多映射**：一个源字段映射到多个目标字段
- **条件映射**：基于条件选择性映射

#### 8.2.2 转换规则
- **数据类型转换**：在不同数据类型之间转换
- **格式转换**：日期、数值等格式的标准化
- **计算规则**：基于表达式进行计算
- **验证规则**：设置字段验证条件
- **自定义转换**：使用自定义脚本进行复杂转换

#### 8.2.3 结构映射
- **嵌套结构映射**：处理嵌套的 JSON 或 XML 结构
- **数组映射**：处理数组和列表数据
- **动态映射**：基于数据内容动态调整映射逻辑

### 8.3 管理映射规则

#### 8.3.1 规则列表管理
在映射规则列表页面，可以：
- 按名称、创建日期、使用频率等排序规则
- 使用筛选器筛选特定类型或行业的规则
- 搜索特定规则
- 批量启用、禁用或删除规则

#### 8.3.2 规则版本控制
系统自动为映射规则提供版本控制功能：
- 记录规则的变更历史
- 支持查看和回滚到历史版本
- 比较不同版本之间的差异
- 保留关键版本作为参考

#### 8.3.3 规则导入导出
- 导出规则为 JSON 或 XML 格式

## 11. 故障排除

### 11.1 常见问题解答

#### 11.1.1 文件上传问题

**问题：上传文件失败，显示「文件格式不支持」**
- **解决方法**：确认文件格式是否在支持列表中；检查文件扩展名是否正确；尝试使用不同的文件格式。

**问题：上传大文件时进度条卡住**
- **解决方法**：检查网络连接是否稳定；减小文件大小或分批上传；调整浏览器超时设置。

**问题：上传成功但文件内容显示不正确**
- **解决方法**：检查文件编码设置；确认文件格式与内容是否匹配；尝试重新保存文件为标准格式。

#### 11.1.2 转换任务问题

**问题：转换任务失败，显示「格式错误」**
- **解决方法**：检查源文件格式是否正确；确认文件结构是否符合预期；尝试修复文件格式问题后重新上传。

**问题：转换结果中字段映射错误**
- **解决方法**：检查映射规则配置；调整字段映射关系；确认数据类型匹配；尝试重新配置映射规则。

**问题：转换任务执行超时**
- **解决方法**：增加任务超时限制；拆分大任务为多个小任务；优化文件内容和结构；减少并行处理任务数量。

#### 11.1.3 系统性能问题

**问题：系统响应缓慢**
- **解决方法**：关闭不必要的浏览器标签和应用；清除浏览器缓存；检查系统资源使用情况；尝试使用私有浏览模式。

**问题：大量数据处理时系统崩溃**
- **解决方法**：增加系统内存分配；减小单次处理的数据量；优化数据结构；检查系统日志获取详细错误信息。
