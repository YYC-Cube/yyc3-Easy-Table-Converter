## 📖 功能介绍

YYC³ Easy Table Converter 的图片尺寸调整工具允许用户轻松修改图片的尺寸，支持多种调整模式和输出格式，适用于网页设计、社交媒体分享、文档嵌入等多种场景。

## 🚀 快速开始

### 基本操作流程

1. **上传图片**：点击上传区域或拖拽图片文件到指定区域
2. **选择调整模式**：
   - **精确尺寸**：直接输入目标宽度和高度
   - **百分比缩放**：按比例放大或缩小图片
3. **设置选项**：
   - 保持宽高比
   - 选择输出格式
   - 调整输出质量
4. **预览效果**：实时查看调整后的图片效果
5. **下载图片**：点击下载按钮保存调整后的图片

## 📐 详细功能说明

### 调整模式

#### 精确尺寸
- 输入具体的宽度和高度（像素）
- 当启用「保持宽高比」时，只需要输入一个维度，另一个维度会自动计算

#### 百分比缩放
- 输入缩放百分比（10% - 500%）
- 图片将按照指定比例等比例缩放

### 高级选项

#### 宽高比设置
- **保持宽高比**：确保图片不会被拉伸变形
- **自由调整**：允许独立设置宽度和高度，可能导致图片变形

#### 输出格式
- **JPEG**：适合照片和复杂图像，文件较小
- **PNG**：支持透明背景，适合图形和图标
- **WebP**：现代图像格式，提供更好的压缩率和质量

#### 质量设置
- 范围：10% - 100%
- 高质量：文件较大但图像细节保留更多
- 低质量：文件较小但可能有画质损失

## 💡 使用示例

### 示例1：调整图片为社交媒体封面图

```
1. 上传原始图片
2. 选择「精确尺寸」模式
3. 设置宽度为1200像素，高度为630像素
4. 启用「保持宽高比」选项
5. 选择输出格式为JPEG
6. 设置质量为90%
7. 下载调整后的图片
```

### 示例2：缩小图片以减少文件大小

```
1. 上传原始高清图片
2. 选择「百分比缩放」模式
3. 设置缩放比例为50%
4. 选择输出格式为WebP
5. 设置质量为80%
6. 下载优化后的图片
```

## ⚠️ 注意事项

1. **文件大小限制**：上传图片大小不超过10MB
2. **支持格式**：支持JPEG、PNG、WebP格式的图片输入
3. **浏览器兼容性**：建议使用Chrome、Firefox、Safari等现代浏览器
4. **质量与文件大小**：更高的质量设置会生成更大的文件

## 📱 移动设备使用

工具完全支持移动设备操作，在手机或平板上同样可以方便地调整图片尺寸：

1. 点击上传区域选择设备中的图片
2. 使用触摸屏调整参数设置
3. 下载按钮会直接将图片保存到设备相册

## 🔄 常见问题

**Q: 调整后的图片质量不理想？**
A: 尝试提高质量设置，或保持宽高比以避免图像拉伸。

**Q: 如何制作正方形图片？**
A: 禁用「保持宽高比」选项，将宽度和高度设置为相同值。

**Q: 可以批量调整多张图片吗？**
A: 当前版本不支持批量处理，需要一张一张调整。
